### Title

MyPortfolio

---

### 内容

MLB や音楽データを使用したページとグラフで可視化した課題管理ツールです。

---

### 使用したライブラリ

React
axios
express
sass
styled-components
react-router-dom
material-ui
etc...

---

### 使用した API

spotify
sportsdata.io

---

### 使用したデータベース

MongoDB

---

### ローカルでの起動

1. GitHub からクローン
2. client 側の起動
   `npm install` --> `npm start`
3. api 側の起動
   `cd api` --> `npm install` --> `npm start`
